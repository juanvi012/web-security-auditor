# Web Security Auditor

Auditor automático de seguridad web desarrollado en Python. Esta herramienta está diseñada para ayudar a administradores y desarrolladores a evaluar la resistencia de sus aplicaciones frente a vectores de ataque comunes.

## 🛠️ Funcionalidades
* **Mapeo Web:** Crawling automatizado de rutas en entornos HTML y PHP.
* **Análisis de Red:** Escaneo integrado de puertos mediante Nmap.
* **Auditoría de Vectores:** Testeo de vulnerabilidades comunes como SQLi, XSS, SSTI, CORS, Open Redirect y fallos de *rate limiting*.
* **Reportes:** Generación de informes detallados con niveles de criticidad y propuestas de mitigación.
